The data include reviews of 43,041 mobile apps. Each app has up to 50 user reviews. Int toal, there are 1,385,607 user reviews.

Each text line represents each app in JSON format. Each app contains the following information: app_id, num_reviews, and a list of reviews.

More details can be found in the paper below.
